# ⚡ 3-Phase Electrical Systems: Star & Delta Visualizer

An interactive, responsive educational web application designed to help engineering students and electrical enthusiasts understand the core concepts, derivations, and mathematical relationships of 3-Phase Star (Y) and Delta (Δ) connections.

🌐 **[Live Demo Link]( https://shivamgohel2008.github.io/three-phase-star-delta-visualizer/)** ---

## ✨ Features

- **Interactive Conceptual Tabs:** Switch seamlessly between Star and Delta configurations with clean, dedicated layouts.
- **Dynamic Vector Visualizations:** High-quality inline SVGs with smooth CSS glow effects and step-by-step mathematical derivations highlighting Kirchhoff's laws ($V_L = \sqrt{3} \times V_{ph}$ and $I_L = \sqrt{3} \times I_{ph}$).
- **Built-in 3-Phase Calculator:** Real-time client-side calculation tool utilizing JavaScript event listeners to compute Line/Phase values and total Active Power ($P$) instantly based on dynamic user input.
- **Responsive Glassmorphism UI:** Modern, responsive dark-mode styling utilizing glassmorphism aesthetics, fluid typography, CSS keyframe animations, and custom scroll-friendly containers for tabular data comparisons.

---

## 🛠️ Tech Stack & Concepts Used

- **Frontend:** HTML5, CSS3 (Modern custom variables, UI animations, flexbox/grid layout systems), JavaScript (ES6 DOM Manipulation and Event Listeners).
- **Mathematics & Physics:** Balanced 3-phase alternating current calculations, vector manipulation, and complex plane trigonometry transformations.
